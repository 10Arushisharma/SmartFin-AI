# 💸 SmartFin AI

> Smarter Budgeting. Better Decisions.

SmartFin AI is an **Agentic AI-powered Personal Finance Management System** that helps users track expenses, manage budgets, predict future spending, detect recurring payments, receive intelligent alerts, and interact with an AI financial assistant.

The project is built using **Python, Streamlit, Pandas, and Machine Learning**, following a modular multi-agent architecture.

# 🏗️ Agentic AI Architecture

SmartFin AI follows a modular multi-agent system:

### ExpenseAnalysisAgent
Analyzes spending data and calculates category-wise expenses.

### DecisionAgent
Evaluates financial status against budget constraints.

### PredictionAgent
Uses machine learning to predict future spending.

### AlertAgent
Generates intelligent financial alerts.

### RecurringExpenseAgent
Detects recurring payments and upcoming bills.

### InsightAgent
Creates personalized financial insights.

### MemoryAgent
Stores historical financial patterns and logs.

### AI Assistant Agent
Provides conversational financial guidance.
